## Hytale-Node-Layout
---

This Project amis to proivde a alternative to the current NodeEditors Auto Position Nodes.

The main goal of this script to preserv the intended Layout of the Author, whilst normalizing and cleaning the graph layout.

---

### Issues

- Smaller nodes might need more Padding to the right in order for the next node to not overlap
